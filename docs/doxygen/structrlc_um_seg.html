Structure to hold a RLC UM PDU segment.

  • lstEnt : This is required for the linked list in which the segments are stored
  • seg : Holds the segment data
  • segSz : The length of the segment in bytes
  • soEnd : SOEnd
  • umHdr : The UM Header for the PDU segment
