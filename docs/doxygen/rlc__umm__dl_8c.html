Handler to discard a SDU.

This function is used to discard a SDU after receiving the Discard Request from the upper layer.The SDU is discarded if it is present and is not mapped to any PDU yet. The discards coming from the upper layer would be coming in sequence according to the sduId, so we should find the sduId at the head of the sduQ. Discards if there is a match else does nothing.

Parameters
[in]rbCbRB control block
[in]sduIdSDU ID of the SDU to be discarded
